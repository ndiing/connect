# Connect

Connect

## Persyaratan

-   [nodejs](https://nodejs.org/dist/v18.16.0/node-v18.16.0-x64.msi)

## Instalasi

Clone project

```bash
git clone https://github.com/ndiing/connect.git
cd connect
```

## Penggunaan

Menjalankan app

```bash
node dist/index.js
```

Memperbarui app

```bash
git reset --hard
git pull
```
